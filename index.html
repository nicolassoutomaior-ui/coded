Python

O que é python?

O Python é uma linguagem de programação de alto nível, versátil e fácil de aprender, conhecida por sua sintaxe limpa e legível. Funciona como uma linguagem interpretada e é usada para uma vasta gama de aplicações, desde o desenvolvimento web e científico até a inteligência artificial.

Como python funciona?

O Python é uma linguagem interpretada, o que significa que o código é executado linha por linha pelo interpretador Python, em vez de ser compilado em código de máquina antes da execução, como acontece com linguagens como C ou Java. Esse processo facilita a depuração e o prototipa rápido.

Sintaxe simples

A estrutura do código Python é projetada para ser intuitiva e próxima da linguagem humana, o que agiliza o desenvolvimento.

Multiplataforma

O mesmo código Python pode ser executado em diferentes sistemas operacionais (Windows, Linux, macOS) sem grandes alterações.

Orientada a objetos

Permite a organização do código em módulos e funções reutilizáveis, facilitando o gerenciamento de projetos complexos.

Para que existe?

O Python existe para simplificar a programação e resolver problemas em diversas áreas. Sua versatilidade a torna uma das linguagens mais usadas no mercado.

Ciência de Dados e Machine Learning

É a linguagem dominante nesse campo, com uma vasta coleção de bibliotecas (como Pandas, NumPy e Scikit-learn) para análise, visualização e cálculos complexos de dados

Desenvolvimento da web

Usada para construir sites e aplicativos robustos com frameworks poderosos como Django e Flask. Empresas como Instagram e Spotify utilizam Python em suas operações de back-end.

Automação de Scripting

Devido à sua simplicidade, é excelente para escrever scripts de automação, simplificando tarefas repetitivas e gerenciamento de sistemas.

Desenvolvimento de Software

Utilizada para o controle de erros, construção automática de software e desenvolvimento de protótipos de aplicações desktop.

Inteligência Artificial e Computação Gráfica

Encontra aplicações em IA, desenvolvimento de jogos e computação gráfica.

Primeiras formas e programação

Plugboards

Nos computadores eletrônicos muito primitivos, como o ENIAC na década de 1940, a programação era feita fisicamente. Os operadores conectavam manualmente cabos e ajustavam chaves em grandes painéis para direcionar os sinais elétricos. Cada configuração de fiação representava um programa ou um conjunto específico de cálculos. Mudar o programa exigia uma religação trabalhosa, o que podia levar dias.

Cartões perfurados

Influenciado pela tecnologia de teares de Jacquard do século XIX e máquinas de tabular da IBM, os cartões perfurados tornaram-se o principal método de entrada e armazenamento de programas.

Como Funcionavam

Os programas eram escritos em papel e, em seguida, perfurados em cartões ou fitas de papel usando máquinas perfuradoras. A presença ou ausência de furos em posições específicas representava dados ou instruções em um formato binário ou alfanumérico codificado. Os cartões eram então lidos por uma máquina, que traduzia os furos em comandos elétricos para o computador. Corrigir um erro significava perfurar um novo cartão e inseri-lo na sequência correta, o que exigia muita organização.

Evolução para linguagens iniciais

Com o tempo, o desenvolvimento de linguagens de programação de baixo nível, como a Linguagem Assembly, permitiu que os programadores usassem mnemônicos (códigos simbólicos) em vez de números binários puros. Um programa chamado "assembler" traduzia esses mnemônicos para o código de máquina que o computador podia executar. Isso, juntamente com o desenvolvimento de linguagens de alto nível como o FORTRAN na década de 1950, tornou a programação mais acessível e menos suscetível a erros manuais.
